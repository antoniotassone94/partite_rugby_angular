# Gestione del calendario delle partite di rugby

**Autore:** Antonio Tassone

**Versione:** 1.0.0

Questa applicazione serve a controllare il calendario delle partite di rugby del campionato americano nella stagione 2022/2023. Per sviluppare questa applicazione è stato utilizzato il linguaggio HTML per i template, il CSS per la grafica e il framework typescript Angular per gestire la logica e le parti dinamiche. Per utilizzare l'applicazione è necessario effettuare il build ed il deploy su un server dedicato che accetta applicazione create con il framework typescript angular.
